# YatraSure — One Booking. Complete Journey. Guaranteed.

India's predictive multi-modal journey planner that eliminates 
last-mile failure using metro schedule data and AI demand prediction.

## 💡 Problem
11 lakh Indians are stranded daily at metro stations — not because 
rides don't exist, but because drivers cherry-pick. YatraSure solves 
this with predictive demand alerts 6 minutes before passengers exit.

## 🛠 Tech Stack
- Frontend: HTML, CSS, Vanilla JavaScript
- AI Layer: Google Gemini API
- Deployment: Vercel
- Driver Network: Namma Yatri Open Source + ONDC/Beckn
